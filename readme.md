# 🌐 Web Traffic Analytics Dashboard

## 📌 Project Overview

The Web Traffic Analytics Dashboard is an interactive data analytics application built using Python, Streamlit, Pandas, and Plotly. The dashboard helps analyze website performance, user engagement, traffic sources, and conversion metrics through interactive visualizations and business insights.

This project enables businesses and digital marketers to understand visitor behavior, identify high-performing traffic channels, improve conversion rates, and make data-driven decisions.

---

## 🚀 Features

### 📊 Key Performance Indicators (KPIs)

* Total Page Views
* Average Session Duration
* Average Bounce Rate
* Average Conversion Rate

### 📈 Traffic Source Analysis

* Compare traffic generated from different sources
* Identify top-performing channels
* Visualize page views by traffic source

### 🎯 Conversion Analysis

* Analyze conversion rates across traffic sources
* Discover the most effective acquisition channels

### ⏱ Session Duration Analysis

* Distribution of session durations
* User engagement insights

### 🚪 Bounce Rate Analysis

* Compare bounce rates across traffic sources
* Identify channels with poor engagement

### 📄 Time on Page Analysis

* Analyze how long users spend on pages
* Understand content effectiveness

### 🔁 Visitor Retention Analysis

* Relationship between previous visits and conversion rate
* Insights into returning visitors

### 🔥 Correlation Matrix

* Understand relationships between website metrics
* Discover factors affecting conversions

### 🤖 Business Insights

* Automatic identification of best-performing traffic source
* Detection of high bounce-rate channels
* Actionable recommendations

## 📂 Dataset Features

The dataset contains the following website analytics metrics:

| Feature          | Description                                     |
| ---------------- | ----------------------------------------------- |
| Page Views       | Number of pages viewed during a session         |
| Session Duration | Duration of the session                         |
| Bounce Rate      | Percentage of visitors leaving after one page   |
| Traffic Source   | Source of website traffic                       |
| Time on Page     | Time spent on a page                            |
| Previous Visits  | Number of previous visits by the user           |
| Conversion Rate  | Percentage of users completing a desired action |

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/ananyaswarnkar19/Web-Traffic-Analysis.git
```

### Navigate to Project Directory

```bash
cd Web-Traffic-Analysis
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
